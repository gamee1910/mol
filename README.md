# MOL - My Own LISP

Documents my journey of learning the C programming language by following the book **Build Your Own Lisp**.

## Learning Resource

Primary resource:

* https://www.buildyourownlisp.com/

# Requirements

* GCC or Clang
* GNU Make
* libedit
* mpc parser library 

## Build

Compile the project:

```bash
make
```

Or compile and run:

```bash
make run
```

## Clean

Remove compiled files:

```bash
make clean
```
